📊 Pandas Data Types Assignment
📌 Overview
This project demonstrates the usage of the select_dtypes() method in the Pandas library using Python. The assignment focuses on selecting and filtering DataFrame columns based on their data types.

🎯 Objectives
Understand DataFrame data types
Learn the select_dtypes() method
Filter numeric, string, and boolean columns
Practice data analysis using Pandas
🛠️ Technologies Used
Python
Pandas
Jupyter Notebook
VS Code
📂 Project Files

▶️ Features
Creates a sample dataset
Displays column data types
Selects numeric columns
Selects string columns
Selects boolean columns
Excludes numeric columns
✅ Conclusion
This assignment demonstrates how Pandas can be used to filter DataFrame columns based on data types using the select_dtypes() method.
